# YuFind
Yudharta Find &amp; Identify Nearby Drops (YuFind!) dirancang untuk membantu user (khususnya mahasiswa Yudharta) dalam melaporkan, mencari, dan mengelola barang hilang maupun ditemukan dalam lingkungan kampus. User dapat mengunggah laporan barang hilang atau ditemukan, dan menelusuri daftar laporan.
